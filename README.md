# Sig — Strict Zig

[![VSCode Extension](https://img.shields.io/badge/vscode-extension-brightgreen)](https://marketplace.visualstudio.com/items?itemName=kikomekobashir.vscode-sig)

[Sig](https://github.com/SB0LTD/sig) support for Visual Studio Code — a capacity-first memory model layer on top of the [Zig](http://ziglang.org/) compiler.

![Syntax Highlighting, Code Completion](./images/example.png)

## Features

- first-class `.sig` file support with syntax highlighting
- install and manage Zig compiler version
- basic compiler linting
- automatic formatting
- Run/Debug programs
- Run/Debug tests
- optional [ZLS language server](https://github.com/zigtools/zls) features
  - completions
  - goto definition/declaration
  - document symbols
  - ... and [many more](https://github.com/zigtools/zls#features)
- full `.zig` and `.zon` support included
